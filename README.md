\# NDM-1 Drug Repurposing Study



\## Overview

This project focuses on identifying potential inhibitors of New Delhi Metallo Beta Lactamase-1 (NDM-1) using molecular docking approaches.



\## Objective

\- To identify zinc-chelating compounds

\- To analyze binding affinity with NDM-1 enzyme



\## Tools Used

\- SwissDock

\- PyMOL 
-swiss ADME
- PLIP


\## Files Included

\- Research\_Report.pdf (final poster/report)

\- ndm1\_results.xlsx (analysis data)

\- images/ (docking and visualization results)



\## Key Findings

\- Identified compounds with strong binding affinity

\- Compared results with known inhibitors
- swiss ADME is run for establishing pharmacokinetic profiles.
